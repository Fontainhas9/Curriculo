Janeiro 2025

Chat Cliente-Servidor

Este código cria um sistema de chat em rede com Python. O servidor corre num socket TCP, aceita múltiplas conexões, guarda os apelidos dos utilizadores e retransmite todas as mensagens recebidas para os clientes ligados. Cada cliente abre uma interface gráfica em Tkinter, onde o utilizador escolhe um nickname, escreve mensagens e visualiza o chat em tempo real. A comunicação entre clientes e servidor é feita por threads, permitindo receber mensagens em paralelo sem bloquear a interface. Assim, vários utilizadores podem conversar em simultâneo num chat local simples.
