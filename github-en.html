Chat Client-Server

This code creates a network chat system with Python. The server runs on a TCP socket, accepts multiple connections, saves user nicknames, and relays all received messages to connected clients. Each client opens a graphical interface in Tkinter, where the user chooses a nickname, writes messages, and views the chat in real time. Communication between clients and the server is threaded, allowing messages to be received in parallel without blocking the interface. This allows multiple users to chat simultaneously in a simple local chat.
